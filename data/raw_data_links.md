# Raw Data Sources

The following public datasets were acquired to generate the feature variables for the Landslide Susceptibility Mapping (LSM) of the Wayanad district.

| Dataset / Variable | Source Agency | Product / Collection ID | Access Link |
| :--- | :--- | :--- | :--- |
| **District Boundary** | Survey of India | Online Maps (State/District Shapefiles) | [Survey of India Online Maps](https://onlinemaps.surveyofindia.gov.in/) |
| **Digital Elevation Model (DEM)** | ISRO (Bhuvan) | CartoDEM Version-3 R1 | [Bhuvan Open Data Archive](https://bhuvan-app3.nrsc.gov.in/data/download/index.php) |
| **Land Use / Land Cover (LULC)** | Google / WRI | `GOOGLE/DYNAMICWORLD/V1` | Accessed via Google Earth Engine |
| **Soil Moisture** | NASA | `NASA/SMAP/SPL4SMGP/007` | Accessed via Google Earth Engine |
| **Rainfall** | IMD Pune | Gridded Rainfall (0.25 x 0.25) NetCDF | [IMD Pune Grid Data](https://www.imdpune.gov.in/cmpg/Griddata/Rainfall_25_NetCDF.html) |
| **Geology & Inventory** (Geomorphology, Lineament, Lithology, Landslide Points) | Geological Survey of India (GSI) | Bhukosh Portal | [GSI Bhukosh](https://bhukosh.gsi.gov.in/Bhukosh/Public) |
